**Krishi Setu** is a Mobile application designed to empower farmers with AI-driven insights and practical tools for efficient farming. The app provides crop suggestions, irrigation guidance, pest management, soil analysis, and regional language support.


## Features ğŸš€

* **AI-Powered Crop Suggestions:** Get personalized recommendations for the best crops to cultivate based on your farm data.
* **Efficient Irrigation Methods:** Learn optimized irrigation techniques to maximize yield and conserve water.
* **Multilingual Support:** Built-in language feature to make the app accessible to farmers in multiple languages.
* **AI Pest & Disease Advisor:** Farmers can describe pests/insects they find on crops, and the app uses AI to suggest possible pest names, causes, and both organic & chemical solutions.
* **AI Soil & Crop Advisor:** Based on soil type, proximity to coastline, nearby water source, and land holding, the app estimates nutrients already present, identifies missing nutrients, and recommends profitable crops.
* **User-Friendly Interface:** Simple and responsive UI for seamless navigation and interaction.

## Installation & Running the App ğŸ’»

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Gopal-Kundu/Krishi-Setu.git
   cd Krishi-Setu
   ```

2. **Frontend Setup**

   ```bash
   cd Frontend
   npm install
   npm run dev
   ```

3. **Backend Setup**
   Open a new terminal and navigate to the backend folder:

   ```bash
   cd Backend
   npm install
   npm start
   ```

4. **Access the App**

   * The app will be running on `http://localhost:5173`
   * Open the link in your browser to explore Krishi Setu.
